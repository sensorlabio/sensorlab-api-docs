Get profile¶

POST /api/v1/profile¶
Synopsis

Get user profile. This can also be used to check authorization token.

Request:

GET /api/v1/profile HTTP/1.1
Host: staging.sensorlab.io
Content-type: application/json

Success response:

HTTP/1.1 200 OK
Content-type: application/json

{
    "email": "test@test.com"
}
Request Headers

  • Authorization – “Bearer: {token}” from authentication.

  • Content-Type – application/json

Status Codes

  • 200 OK – No errors.

  • 401 Unauthorized – Wrong authorization token

Note

Available for:

  • User token
